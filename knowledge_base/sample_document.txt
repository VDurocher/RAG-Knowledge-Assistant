RAG Knowledge Assistant — Sample Document
=========================================

Welcome to the RAG Knowledge Assistant. This sample document demonstrates how the system works.

## How to Use

1. Add your company documents (PDF or TXT) to this knowledge_base/ folder.
2. Launch the application with: streamlit run app/main.py
3. Ask questions in the chat interface.
4. The assistant will answer based on your documents and show citations.

## Supported File Types

- PDF files (.pdf) — invoices, contracts, reports, manuals
- Text files (.txt) — policies, guidelines, notes
- Markdown files (.md) — documentation, wikis

## Example Questions

Once you have added your documents, you can ask questions like:
- "What is our refund policy?"
- "How do I submit an expense report?"
- "What are the onboarding steps for new employees?"
- "What are the Q4 revenue figures?"

## Configuration

The system supports two embedding modes:
- Local mode (default): Uses HuggingFace sentence-transformers, completely free and offline.
- OpenAI mode: Uses text-embedding-3-small for higher accuracy on complex documents.

Set EMBEDDER_TYPE=openai in your .env file to switch modes.

## Data Privacy

All documents remain on your local machine. Nothing is sent to external servers
except the question text and retrieved passages when using OpenAI for generation.
For fully local operation, consider integrating a local LLM (Ollama).
